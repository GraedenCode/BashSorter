# BashSorter
This script automatically sorts images and text files into their own folders or creates a folder for them.

# Features

- Sorts '.txt', '.jpg', '.jpeg', '.png'.
- Logs each run to 'logs/'
- Can be scheduled in cron

# Usage

.sorter.sh /path/to/folder
